CSS Typography Introduction

Types of TypeFaces

  1. Serif
  2. Sans Serif
  3. Monospace - giving equal space for all character
  4. Handwriting/Cursive - looks like hand written
  5. Display/Fantasy

Variation

  1. weight - related to thickness of the word or letter
  2. style - defining the style like italic, bold etc
  3. stretch- related to width of the letter, increasin the width- extended font, decreasing the width- condensed fonts

Font size units

  1. Pixels
  2. Percentages
  3. EMs
  4. REMs

Font weight
